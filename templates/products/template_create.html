Information
Product Template

A product template represents a style or design that can have multiple variants (different sizes, colors, etc.). Each template has a unique StyleCode.

StyleCode

The StyleCode is a unique identifier for the product template. It's used to group related variants together.

GST Information

HSN code and GST tax group are required for GST compliance. These will be inherited by all variants of this template.

Next Steps

After creating the template, you can:

  • Add product images
  • Configure GST rates
Tips
  • Use descriptive product names
  • StyleCode should be unique
  • Select appropriate HSN code
  • Choose correct GST tax group
  • Add relevant tags for filtering
